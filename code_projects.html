My Programming Projects A brief description of some of the programming projects I have done.

Concon

The Android version of Concon, an app to help gamers and geeks and nerds find the next great convention to attend. Features anime, gaming, sci-fi, horror, and more.
I improved the UI (not shown here) and implemented features that allowed the user to filter and search through the list of conventions. I also added sharing features, and exporting to other applications.
This was written in Android Studio using the Android sdk.

Reek Rank

Essentially Yelp for bathrooms. This web application helps users find, review, and rate nearby bathrooms.
This was a project for my Software Developement class in the fall of 2016 and was written with javascript frontend using bootstrap, jquery, and google maps. The backend was written with Java and SQL and uses Servlets to send data.

Jeopardy

A fully-fledged, networked Jeopardy game. This is a Java desktop application for my Software Developement class written in Eclipse with Java Swing. This game lets users play locally, or host/join a networked game with other players. Also includes various timers and animations.

Cucco Run

A simple side-scrolling game where the user plays as the hero of time and must jump over Cuccos as they fly across the screen. The further the player runs, the larger and more frequent the Cuccos will become.
I wrote this as part of my Android Internship in the summer of 2014

I ported this as a web-app in the Spring of 2017 using javascript and the Phaser physics engine.

It can be found here.

iOS and Android Clones

Tumbleclone

A basic Tumblr dashboard app that lets users log in with their Tumblr account and see their most recent dashboard posts. Allows for filtering of post types and pull to refresh.
This was written as part of a take-home excercise for the Tumblr interview process.

Talkies

Uses the IMBD movie database to show the user a list of the latest movies in gridview.
Written as part of a iOS tutorial

Tipsy Tips

Helps users calculate tip costs using current local currency.
Written as part of a iOS tutorial

Also as part of the iOS tutorial I have written simplified clones of Tumblr and Instagram, and a basic iOS messenging application.

Verilog Projects

Various circuit designs created using Xilinx design software and implemented on an FPGA board.

Vending Machine

A basic circuit system that simulates vending machine functionality. Written as part of my Digital Systems Design class

Square Root Calculator

Lets users input a value, and then uses a square-root algorithm to calculate and output the square root. Written as part of my Digital Systems Design class

FPGA Tetris

Implemented a Tetris game as project for my Intro to Digital Circuits class. Used Verilog to output to a monitor through VGA output, and designed the game logic with a state machine.
